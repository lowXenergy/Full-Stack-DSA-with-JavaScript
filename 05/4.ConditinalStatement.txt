## LEVEL 1: EASY — CONTROL FLOW BASICS

1. What is the purpose of `if` and `else` statements in JavaScript?
the if and else statements are used to control the flow of data based on some conditions in our JavaScript. The else statement will only execute if the if statement does't execute.

2. When do we use `else if` instead of multiple `if` statements?
We use else if when we have to make decision according to the given multiple conditions.

3. What happens if none of the `if` / `else if` conditions match and there is **no `else`**?
Any code written in `if` / `else if` conditions will not be executed.

14. When should you prefer **early return** instead of deep `if–else` nesting?
You should prefer **early return** when handling invalid conditions or edge cases first, to avoid deep `if–else` nesting and keep the function clean, readable, and easy to maintain.

19. Design logic for an **e-commerce discount system**:

    * Cart value < 500 → No discount
    * 500–1000 → 5% discount
    * 1000–5000 → 10% discount
    * > 5000 → 20% discount

function discount() {
    let amt = prompt("Enter the amount");

    if (amt === null) return "NO SHOPPING DONE";
    if (amt.trim() === "") return "NO SHOPPING DONE";

    let value = Number(amt);

    if (isNaN(value)) return "Not a number";
    if (value < 500) return "No discount";
    if (value <= 1000) return "5% discount";
    if (value <= 5000) return "10% discount";
    return "20% discount";
}

console.log(discount());

 function validation() {
    let name = prompt("Enter your name");
    if (name === null) return "Name input cancelled";
    if (name.trim() === "") return "Name is required";

    let email = prompt("Enter your email");
    if (email === null) return "Email input cancelled";

    return "Form validated";
}
